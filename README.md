## 🚀 Overview

This portfolio is a high-performance, single-page application built with a focus on **pixel-perfect design**, **responsive interactions**, and **multi-language support**. It highlights my architectural philosophy: From Neural Data to Pixel Perfection.

## ✨ Key Features

- 🌓 **Dynamic Theme Switching**: Seamless transition between Dark and Light modes with system preference detection.
- 🌍 **Multi-Language Support**: Full i18n support for English, Turkish, and French.
- 📱 **Mobile First**: Fully responsive design with a custom-built mobile navigation system.
- ⚡ **Performance Optimized**: Lightweight assets, smooth animations (CSS & JS), and high accessibility scores.
- 🛡️ **Clean Architecture**: Modular code structure across HTML, CSS, and Vanilla JavaScript.

## 🛠 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Typography**: Inter & Outfit (Google Fonts)
- **Deployment**: GitHub Pages

## 📂 Project Structure

- `index.html`: Core structure and semantic markup.
- `style.css`: Comprehensive design system, animations, and responsive layouts.
- `script.js`: i18n engine, theme management, and interactive UI logic.
- `assets/`: Optimized images and media components.

## ⚙️ Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ismailcelik-tr/ismailcelik-tr.github.io.git
   ```
2. Navigate to the directory:
   ```bash
   cd ismailcelik-tr.github.io
   ```
3. Open `index.html` in your favorite browser.
